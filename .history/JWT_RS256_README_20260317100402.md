# JWT RS256 Implementation & Change Password Feature

## Các Thay Đổi Được Thực Hiện

### 1. **Chuyển đổi JWT từ HS256 sang RS256**

#### File: `private.pem` & `public.pem`
- Tạo cặp khóa RSA 2048-bit
- **private.pem**: Dùng để ký (sign) JWT tokens
- **public.pem**: Dùng để xác minh (verify) JWT tokens

#### File: `routes/auth.js`
```javascript
// Thay đổi:
// Cũ: jwt.sign({id: result._id}, 'secretKey', {algorithm: 'RS256'})
// Mới: jwt.sign({id: result._id}, privateKey, {algorithm: 'RS256'})

const privateKey = fs.readFileSync('private.pem', 'utf-8');
let token = jwt.sign({id: result._id}, privateKey, {
    expiresIn: '1d',
    algorithm: 'RS256'
})
```

#### File: `utils/authHandler.js`
```javascript
// Thay đổi:
// Cũ: jwt.verify(key, 'secretKey')
// Mới: jwt.verify(key, publicKey, {algorithms: ['RS256']})

const publicKey = fs.readFileSync('public.pem', 'utf-8');
let result = jwt.verify(key, publicKey, {algorithms: ['RS256']})
```

### 2. **Thêm Hàm Change Password**

#### File: `routes/auth.js`
```javascript
router.post('/changepassword', CheckLogin, ChangePasswordValidator, validationResult, async function (req, res, next) {
    try {
        let { oldPassword, newPassword } = req.body;
        let user = req.user;
        
        // Kiểm tra mật khẩu cũ
        if (!bcrypt.compareSync(oldPassword, user.password)) {
            res.status(403).send({ message: "mat khau cu sai" });
            return;
        }
        
        // Cập nhật mật khẩu mới
        user.password = newPassword;
        await user.save(); // Schema sẽ tự hash
        
        res.send({ message: "doi mat khau thanh cong" });
    } catch (err) {
        res.status(400).send({ message: err.message });
    }
})
```

#### File: `utils/validatorHandler.js`
```javascript
ChangePasswordValidator: [
    body('oldPassword').notEmpty().withMessage("mat khau cu khong duoc de trong"),
    body('newPassword').notEmpty().withMessage("mat khau moi khong duoc de trong")
        .bail()
        .isStrongPassword(options.password)
        .withMessage(`password dai it nhat 8 ki tu...`)
]
```

**Validation Requirements:**
- `oldPassword`: Bắt buộc, phải khớp với mật khẩu hiện tại
- `newPassword`: Bắt buộc + Mật khẩu mạnh
  - Tối thiểu 8 ký tự
  - Ít nhất 1 chữ hoa
  - Ít nhất 1 chữ thường  
  - Ít nhất 1 số
  - Ít nhất 1 ký tự đặc biệt

## Features Mới

### Endpoints:
1. ✅ `POST /api/v1/auth/register` - Đăng ký user
2. ✅ `POST /api/v1/auth/login` - Đăng nhập, nhận JWT RS256
3. ✅ `GET /api/v1/auth/me` - Lấy info người dùng (cần token)
4. ✅ `POST /api/v1/auth/changepassword` - Đổi mật khẩu (cần token)
5. ✅ `POST /api/v1/auth/logout` - Logout

### Security Improvements:
- JWT dùng RS256 (asymmetric) thay vì HS256 (symmetric)
- Key size 2048 bits
- Mật khẩu mới phải validate (strong password)
- So sánh mật khẩu cũ với bcrypt
